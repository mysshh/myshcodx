# myshscodx

myshscodx is a custom interpreted programming language inspired by fantasy-themed keywords.

## Features

- Dracarys (variable declaration)
- Rytsas (output)
- Zaldrizes (conditional statements)
- MunaZaldrizoti (else branch)
- Valonqar (for loops)
- Darilaros (end block)

## Example

```text
Dracarys score = 95

Zaldrizes score >= 90
Rytsas "Dragon Master"

MunaZaldrizoti
Rytsas "Keep Training"

Darilaros
```

## Interpreter

The language is implemented using a Python interpreter.
